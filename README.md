## 🔗 Integration with Home Assistant

ZTE-ModemFlow works by sending your modem data to Home Assistant.

To link the two:

1. Open your Home Assistant dashboard.
2. Navigate to **Configuration > Integrations**.
3. Add a new integration named "ZTE-ModemFlow".
4. Follow the simple prompts inside Home Assistant to connect it with the app.
5. After setup, you can see modem stats and control options in your Home Assistant interface.

## 🛠️ Troubleshooting Tips

- If the app cannot connect to the modem, check your network cables and modem power.
- Verify the modem IP address is correct (default is usually `192.168.1.1`).
- Use the modem’s web interface in your browser to confirm login credentials.
- Make sure your Windows firewall allows ZTE-ModemFlow to communicate.
- Restart the app if it becomes unresponsive.
- Update the app regularly by checking the Releases page.
